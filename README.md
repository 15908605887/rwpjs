## 说明

RPW.JS 是一套 React 中 web网页开发的全栈解决方案。支持以下的内容

- 通过脚手架快速构建后端管理系统, 通过 `@rwp/cli` 提供 
- 40 + 的UI 组件, 通过 `@rwp/react-ui` 提供, 高性能组件,例如表格的虚拟滚动,以及无限滚动
- 微前端架构, 通过注册中心引入其他的前端模块(PS: 在开发中)
